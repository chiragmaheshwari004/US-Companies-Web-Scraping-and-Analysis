# US Companies Web Scraping and Analysis

## Project Overview

This project demonstrates the process of collecting, cleaning, analyzing, and visualizing data using Python. Company data was scraped from Wikipedia using Requests and BeautifulSoup, transformed into a structured dataset using Pandas, and analyzed through exploratory data analysis and visualizations.

## Objectives

* Extract company information from a public website.
* Convert unstructured HTML data into a structured dataset.
* Perform data cleaning and transformation.
* Analyze company revenue, employee count, and industry distribution.
* Create visualizations to identify business insights.

## Tools and Technologies

* Python
* Requests
* BeautifulSoup
* Pandas
* Matplotlib
* Jupyter Notebook

## Project Workflow

1. Scraped company data from Wikipedia.
2. Parsed HTML content using BeautifulSoup.
3. Stored the extracted data in a Pandas DataFrame.
4. Cleaned and transformed the dataset.
5. Exported the data to CSV format.
6. Performed exploratory data analysis.
7. Created visualizations and extracted insights.

## Key Analysis Performed

* Top companies by revenue
* Top companies by employee count
* Industry distribution analysis
* Revenue comparison across companies

## Data Source

Wikipedia – List of Largest Private Companies in the United States
